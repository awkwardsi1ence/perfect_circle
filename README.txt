## Запуск программы

1. Установите Python, если его еще нет на вашем компьютере.
2. Скачайте или склонируйте репозиторий с игрой.
3. Откройте командную строку (терминал) и перейдите в папку с игрой.
4. Установите зависимости, выполнив команду: `pip install pygame`
5. Запустите игру, выполнив команду: `python main.py`

## Зависимости

- Python 3.x
- Pygame
